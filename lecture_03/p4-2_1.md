# Programming exercise 1 (ch. 4.2)

Assuming x is 15.0 and y is 25.0, what are the values of the followong conditions?

```c
int main(void) {
  float x = 15.0, y = 25.0;

  x != y;

  x < x;

  x >= y - x;

  x == y + x - y;
}
```
